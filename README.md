# Playground-Terraform-Azure

A collection of Terraform exercises for learning Azure infrastructure deployment.

## Exercises

### [Exercise 1 - Single VNet with Virtual Machine](./Ex1/)
Deploy a single Ubuntu VM in an Azure Virtual Network.

**Learn:** Virtual Networks, Subnets, Virtual Machines

### [Exercise 2 - Load Balancer with Virtual Machines](./Ex2/)
Deploy two VMs behind an Azure Load Balancer for high availability.

**Learn:** Load Balancing, Traffic Distribution, High Availability

## Quick Start

1. **Login to Azure**
   ```bash
   az login
   ```

2. **Navigate to an exercise**
   ```bash
   cd Ex1/  # or Ex2/
   ```

3. **Follow the exercise README**
   Each exercise has detailed instructions in its own README file.
